# Aditya Srivastava — Interactive 3D Portfolio City

🌐 **Live Demo:** https://adityasrivastava-dev.github.io/

An immersive 3D portfolio where you **drive through a living city**, and every building represents a real backend system I’ve designed and built over 4+ years as a Backend Architect.

This is not a static portfolio — it’s an **interactive experience** built entirely in the browser.

---

## 🧠 Concept

Instead of listing projects, this portfolio turns them into a **navigable world**.

- Each building = a real backend system
- Exploration replaces scrolling
- Interaction replaces reading

👉 _Experience the systems instead of just reading about them._

---

## 🚀 Features

### 🌆 Interactive World

- Fully explorable 3D city
- 12 buildings mapped to real production systems
- District-based layout

### 🚗 Physics-Based Driving

- Smooth acceleration, braking, and steering
- Speed-sensitive handling
- Collision-aware movement

### 🎥 Cinematic Camera System

- Dynamic follow camera
- Smooth transitions and focus states
- Speed-based FOV adjustments

### 🌦️ Dynamic Weather System

- Day / Night / Sunset / Fog / Rain / Snow
- Each mode changes lighting, fog, and atmosphere

### 🏛️ Building Interaction

- Proximity detection
- Enter buildings to explore project details

Two modes:

- **Recruiter Mode** → business impact
- **Engineer Mode** → technical depth

### 🤖 Oracle AI Assistant

- Ask questions about any system
- Powered by Claude (Anthropic API)

### 📖 Career Journey Board

- Timeline from 2015 → present

### 🎵 Generative Audio

- Ambient soundscape using Web Audio API

## 🎮 Controls

| Key           | Action                 |
| ------------- | ---------------------- |
| W / ↑         | Accelerate             |
| S / ↓         | Brake / Reverse        |
| A / D / ← / → | Steering               |
| E             | Enter nearest building |
| Space         | Handbrake              |
| J             | Journey Board          |
| T             | Change Weather         |

## 🛠️ Tech Stack

- **Three.js (r128)**
- **Vanilla JavaScript (No frameworks, no build tools)**
- **Web Audio API**
- **Anthropic Claude API (Oracle AI)**
